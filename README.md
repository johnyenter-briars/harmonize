# Harmonize

### The Goal 

Content should be free and open. I dream of this future, naive as it is. 

### Complete Feature List
- [X] Integration with [QBT](https://www.qbittorrent.org/download)
- [X] Async background job monitoring / history
- [X] Integration with [Kodi's](https://kodi.wiki/view/Main_Page) [JSON RPC API](https://kodi.wiki/view/JSON-RPC_API)
    - [X] Multi-media control of Kodi 
    - [X] File transfer + delete from Kodi video library
- [X] Season and individual episode management
- [X] Sub title tracking (.srt files)
- [X] Integration with [TVC](https://github.com/johnyenter-briars/tvc) to control TV [libcec](https://github.com/Pulse-Eight/libcec)
